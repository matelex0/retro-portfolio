## 📼 About The Project

**Retro Portfolio** is a creative, retro-styled developer portfolio website featuring a Bento grid layout, interactive elements, and a playable Snake game. Built with vanilla web technologies, it offers a unique way to showcase your work.

It is designed to be easily customizable via a configuration file, allowing developers to personalize their portfolio without diving deep into the code.

> "Simplicity is the ultimate sophistication."

---

## ✨ Features

- **🎨 Retro & Cyberpunk Aesthetic**: CRT scanline effects, glowing accents, and pixel fonts.
- **🍱 Bento Grid Layout**: Clean, responsive, and organized content presentation.
- **🐍 Interactive Snake Game**: A fully playable "SNAKE.EXE" game embedded directly in the dashboard.
- **⚙️ Dynamic Configuration**: Easily customize content (profile, colors, projects, tech stack) using `config.yaml` without touching the code.
- **💼 Project Showcase**: A modal window to display your work, skills, and timeline.
- **🕒 Real-time Elements**: Live local time display.
- **📱 Fully Responsive**: Optimized for desktop, tablet, and mobile devices.

## ⚙️ Configuration

You can personalize this portfolio by editing the `config.yaml` file. No coding knowledge required!

### Customizable Options:
- **Profile**: Name, role, bio, location, and status.
- **Contact**: Email address.
- **Colors**: Theme colors (backgrounds, accents, text).
- **Stack**: List of technologies you want to display.
- **Projects**: Add your own projects with titles, descriptions, images, and links.

**Example `config.yaml` snippet:**
```yaml
profile:
  name: "YOUR NAME"
  role: "Your Role"
  
colors:
  accent: "#d651ffc4" # Change the main accent color
```

## 🚀 Getting Started

To run the portfolio locally:

1. **Clone the repository**
   ```bash
   git clone https://github.com/matelex0/retro-portfolio.git
   ```

2. **Navigate to project directory**
   ```bash
   cd retro-portfolio
   ```

3. **Start a local server** (Python example)
   *Note: Required for YAML configuration to load correctly due to CORS.*
   ```bash
   python -m http.server 8000
   ```

4. **Explore**
   Open `http://localhost:8000` in your browser.
